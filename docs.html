WisdomBot Dokumentation

Hier findest du alles, was du über die Einrichtung und Nutzung von WisdomBot wissen musst.

Erste Schritte

Um WisdomBot zu deinem Server hinzuzufügen, klicke hier auf den Einladungslink. Stelle sicher, dass der Bot die notwendigen Berechtigungen (Administrator oder spezifische Berechtigungen) hat.

Standard-Präfix

WisdomBot nutzt standardmäßig Slash-Commands (/) für alle Funktionen. Das Legacy-Präfix ist !, wird aber nicht mehr aktiv empfohlen.

Wichtige Befehle

/help

Zeigt eine Liste aller Befehle an oder gibt spezifische Informationen zu einem Befehl.

/config

Öffnet das Server-Dashboard zur Konfiguration von Moderation, Willkommensnachrichten und mehr.

/mod ban [user] [reason]

Verbannt einen Nutzer vom Server. (Moderationsberechtigungen erforderlich).

/fun joke

Erzählt einen zufälligen Witz. Spaß für die Community!
